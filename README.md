# 📚 Doc2Quiz — Document → Quiz Generator (Hackathon Project)

Instant **document-to-quiz converter** built during the **VibeCode Hackathon 2025** —  
upload any study document and get **auto-generated MCQs & quizzes**, powered by  
**HTML + Tailwind + JavaScript (Frontend)** and **n8n Workflow Automation (Backend).**

---

## 🚀 Features

- 📄 Upload PDFs, Docs, Notes & Text
- 🤖 Smart quiz generation using automated workflows
- ⚡ Instant output — no manual question creation
- 🎯 MCQs, True/False & Short Questions supported
- 💻 Clean & responsive UI with Tailwind CSS
- 🔗 Backend automation via n8n workflows

---

## 🧠 How It Works

1️⃣ User uploads a document  
2️⃣ Text is extracted and sent to n8n workflow  
3️⃣ Workflow processes the content using LLM & generators  
4️⃣ Generated quiz returns to frontend and displayed instantly

---

## 🛠 Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | HTML, Tailwind CSS, JavaScript |
| Automation Backend | n8n Workflow Automation |
| Parsing / AI | LLM APIs (configurable) |
